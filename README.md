# Discord Mriamys Bot 🚀

Ультимативный бот для Discord сервера **Mriamys**, объединяющий систему экономики, развлечений, музыки и автоматизации ролей.

## 🌟 Основные функции

### 💰 Экономика и Магазин
*   **VibeКоины**: Внутренняя валюта сервера, которую можно заработать за активность в чате и голосе.
*   **🛒 Магазин VibeCity**: Интерактивная панель для покупки предметов:
    *   🏷️ **Погоняло**: Смена ника любому участнику на 1 час.
    *   🎭 **Фейковый статус**: Кастомная приписка к нику.
    *   ⚡ **Буст XP**: Удвоение получаемого опыта на 2 часа.
    *   🔊 **Рандомный высер**: Бот заходит в ваш голосовой канал и проигрывает случайные аудио-мемы в течение часа.

### 🎮 Игровые комнаты (Threads)
Все игры запускаются в персональных приватных ветках (Threads), чтобы не засорять основные чаты:
*   🎰 **Казино**: Классические ставки на коины.
*   📦 **Кейсы**: Открытие коробок с ценными (и не очень) призами.
*   🃏 **Блэкджек**: Карточная игра против дилера с интерфейсом на кнопках.
*   ⚔️ **Дуэли**: Сражения между участниками на VibeКоины.
*   💡 **Викторина (КХСМ)**: Интеллектуальное шоу "Кто хочет стать миллионером" (Соло или Дуэль) с базой более чем на 30,000 вопросов.

### 📈 Прогресс и Уровни
*   **Система уровней**: Опыт начисляется за сообщения и время в голосовых каналах.
*   **Профиль**: Красивая карточка профиля (`/profile`) со статистикой, уровнем и балансом.
*   **Лидеры**: `/top` по уровням и богатству.
*   **Квесты и Достижения**: Награды за активность и игровые успехи.

### 🎵 Музыкальный плеер
*   Поддержка **YouTube** и **Spotify**.
*   Управление через кнопки и слеш-команды (`/play`, `/skip`, `/queue`, `/stop`).

### 🎥 Автоматизация Стримеров
*   **Авто-роль**: Бот автоматически выдает роль `[🎥] Стример` при запуске трансляции (через Discord Go Live / Twitch).
*   **Контроль активности**: Если стример не выходил в эфир более **30 дней**, роль автоматически снимается.
*   **Уведомления**: Интеграция с Twitch для отслеживания статуса.

### 🎭 Выдача ролей
*   Интерактивные панели (`/setup_roles`) для выбора:
    *   **Игровых ролей**: Открывают доступ к скрытым категориям чатов конкретных игр.
    *   **IT ролей**: Роли для программистов и вайбкодеров.

### 🔔 Администрирование
*   **Telegram Notifications**: Уведомления администратору о важных событиях сервера.
*   **Логирование**: Запись действий модерации и событий сервера.
*   **Welcome Cards**: Генерация приветственных изображений для новых участников.

---

## 🛠 Технологии
*   `discord.py` (v2.x)
*   `MySQL` + `aiomysql`
*   `Easy-PIL` (графика)
*   `yt-dlp` + `ffmpeg` (музыка)
*   `deep-translator` (перевод вопросов викторины)

---

## 🚀 Установка и запуск

1.  Установите зависимости: `pip install -r requirements.txt`.
2.  Настройте `.env` файл (Токен бота, БД, Telegram ID).
3.  Запустите бота: `python main.py`.
