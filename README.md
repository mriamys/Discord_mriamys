# Discord Mriamys Bot 🚀

Ультимативный многофункциональный бот для Discord сервера **Mriamys**. 
Бот объединяет в себе продвинутую экономику, развлечения, музыку, интеграцию со стриминговыми платформами и мощную систему автоматизации.

## ✨ Основные возможности

### 🎮 Развлечения и Игры
*   🎰 **Казино и Кейсы**: Испытай удачу в открытии кейсов или ставках в казино.
*   🃏 **Блэкджек**: Полноценная карточная игра с интерфейсом на кнопках.
*   ⚔️ **Дуэли**: Вызывай других участников на битву за VibeКоины.
*   🧠 **Викторина (КХСМ)**: Игра "Кто хочет стать миллионером" с огромной базой вопросов из локального CSV-файла.
*   🔊 **Аудио-мемы**: Быстрый запуск легендарных звуковых мемов в голосовой канал.

### 💰 Экономика и Прогресс
*   🪙 **VibeКоины**: Глобальная валюта сервера. Зарабатывай, играй и трать!
*   🏪 **Магазин**: Покупка ролей, предметов и уникальных перков.
*   📈 **Уровни и Опыт**: Система уровней с кастомными карточками профиля (генерация через `PIL`).
*   🏆 **Достижения и Квесты**: Выполняй задания и получай уникальные награды и ачивки.

### 🎵 Музыкальный плеер
*   Поддержка **YouTube** и **Spotify** (автоматический поиск треков).
*   Удобное управление через кнопки и слеш-команды.
*   Высокое качество звука через `yt-dlp` и `ffmpeg`.

### 🛡️ Автоматизация и Модерация
*   🎥 **Стримеры**: 
    *   Автоматическая выдача роли при старте трансляции.
    *   **Smart Cleanup**: Автоматическое снятие роли "Стример" после 30 дней неактивности.
    *   Интеграция с Twitch API для отслеживания статуса.
*   🔊 **Динамические комнаты**: Создание временных приватных каналов ("Приватка +").
*   🎭 **Роли**: Система выбора ролей через интерактивное меню (Self-roles).
*   📜 **Логирование**: Детальные логи действий в специальные каналы.

### 🔔 Уведомления
*   Интеграция с **Telegram**: Администратор получает уведомления о критических событиях и бэкапах прямо в мессенджер.

## 🛠 Стек технологий
*   **Ядро**: `python 3.10+`, `discord.py`
*   **БД**: `MySQL` (через `aiomysql`)
*   **Медиа**: `yt-dlp`, `ffmpeg`, `spotipy`
*   **Графика**: `Easy-PIL` (для карточек профиля)
*   **Интеграции**: `Twitch API`, `Telegram Bot API`

## 🚀 Быстрый старт

### 1. Подготовка окружения
Убедитесь, что у вас установлен `ffmpeg` и `MySQL`.

```bash
# Ubuntu/Debian
sudo apt update && sudo apt install ffmpeg python3-pip mysql-server -y
```

### 2. Установка
```bash
git clone https://github.com/mriamys/Discord_mriamys.git
cd Discord_mriamys
pip install -r requirements.txt
```

### 3. Настройка параметров (`.env`)
Создайте файл `.env` в корневой папке:
```env
DISCORD_TOKEN=your_discord_bot_token

# База данных
DB_HOST=localhost
DB_USER=mriamys
DB_PASS=your_db_password
DB_NAME=mriamys_bot

# Уведомления Telegram
TELEGRAM_BOT_TOKEN=your_tg_bot_token
TELEGRAM_ADMIN_ID=your_tg_id
ADMIN_DISCORD_ID=your_discord_id

# Опционально: Twitch
TWITCH_CLIENT_ID=your_id
TWITCH_CLIENT_SECRET=your_secret
```

### 4. Необходимые файлы
Для корректной работы некоторых функций убедитесь в наличии:
*   `База КХСМ.csv` — База вопросов для викторины.
*   Папка `audio/memes` — Локальные файлы мемов (если сайт MyInstants недоступен).
*   Папка `assets` — Шрифты и фоны для генерации карточек профиля.

### 5. Запуск
```bash
python main.py
```

Для работы в режиме 24/7 рекомендуется использовать `pm2` или `systemd`.
